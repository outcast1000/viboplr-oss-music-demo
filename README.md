# Open Source Test — Viboplr music source (demo)

A minimal, royalty-free demo of a **Viboplr manifest music source**. The audio
files are synthesized sine tones (generated with ffmpeg) — no copyrighted material.

## Add to Viboplr

Open [the landing page](https://outcast1000.github.io/viboplr-oss-music-demo/)
and click **▶ Add to Viboplr**, or click this link directly:

```
viboplr://add-collection?kind=manifest&name=Open%20Source%20Test&url=https%3A%2F%2Fraw.githubusercontent.com%2Foutcast1000%2Fviboplr-oss-music-demo%2Fmain%2Fmanifest.json
```

Viboplr confirms before subscribing, then ingests the tracks from
[`manifest.json`](manifest.json) into a `manifest`-kind collection. They appear
in search / Home and stream directly from the URLs below.
